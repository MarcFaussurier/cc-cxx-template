# C-CPP-starter

A basic repo I made so that I will spend less time in creating my next projects.

## Get started

1) git clone https://github.com/MarcFaussurier/C-CPP-starter.git yourproject
2) cd yourproject && make install

## Comands

- make
- make run
- make test 
- make watch 
- make watch-test 
- make watch-run
- make update
- make clean
- make fclean
- make release
- make norme
